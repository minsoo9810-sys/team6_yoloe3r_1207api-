# YOLOE3R: Perception-Efficient 3D Reconstruction


### Quick Start
#### Install
```bash
conda create --name yoloe3r python=3.10
conda activate yoloe3r
git clone 
pip install requirements.txt
```
#### Usage
```bash
python pe3r_demo.py
```
